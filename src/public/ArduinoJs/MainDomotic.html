Control systems of our web portal

1Control by Microcontroller System

The section of the plant related to this Method, will allow you to directly control the PLC system using a Microcontroller attached directly to the analog voltage inputs of the SCADA, for this it uses full duplex serial communication, the Firmata communication protocol and the Jhonny-Five FrameWork, between the central computer and the SCADA. This method allows the PLC to be manipulated directly using ON/OFF controlled by the MCU and commanded by a use in the graphical web interface of the page.

In order to initialize this control, you must note that the MCU is properly connected to the computer, the computer has proper load power, and the Enterprise server is initialized with the hosting server communication protocols.

2Control by LabVIEW HDMI System

In this section ofthe page you will be able to control the production of several SCADA systems at the same time by implementing an HDMI generated in LabVIEW, which is linked by the OPC protocol to the PLC for control.

To initialize this form of control, you should be aware that the server of the respective program in LabVIEW is initialized, in turn, because of Plugin compatibility, you must use the Internet Explorer Browser.

3Industrial Process by LabVIEW HDMI

In this section of the plant you can see the base architecture of the industrial process of our company, where you can see the PID model, the HDMI control of LabVIEW and the general process of the plant by Grafcet.
